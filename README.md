# User Management Dashboard

A React-based User Management Dashboard built using Vite and Axios.

## Features

- View Users
- Add User
- Edit User
- Delete User
- Search Users
- Filter Users
- Sort Users
- Pagination (10,25,50,100)
- Responsive UI
- Client-side Validation
- Error Handling

## Tech Stack

- React
- Vite
- Axios
- CSS

## API

https://jsonplaceholder.typicode.com/users

## Installation

```bash
npm install
```

Run

```bash
npm run dev
```
